# Fraud Detection & Risk Analytics Dashboard

## Overview
This project focuses on analyzing financial transaction data to identify suspicious transaction patterns, fraud trends, and high-risk customer behaviors using SQL and Power BI.

---

## Tools Used
- SQL
- SQLite
- Power BI
- CSV Dataset

---

## Key Features
- Fraud trend monitoring
- Geographic fraud analysis
- Payment method risk analysis
- Device-based fraud analysis
- KPI dashboards
- Interactive slicers and filters

---

## Dashboard KPIs
- Total Transactions
- Fraud Cases
- Fraud Percentage
- Average Fraud Amount

---

## Key Insights
- High-value transactions showed higher fraud occurrence.
- Credit card transactions had elevated fraud frequency.
- Fraud activity varied across cities and device types.
- Interactive dashboards improved fraud visibility and analysis.

---

## Files Included
- SQL queries
- SQLite database
- CSV dataset
- Power BI dashboard
- Dashboard screenshots

---

## Project Outcome
Built an end-to-end fraud analytics solution for identifying suspicious transaction patterns and improving analytical fraud monitoring workflows.
